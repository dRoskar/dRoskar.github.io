Holistic

Forest fire prevention, surveilance and intervention system

Homepage

Features

  • live map
  • live fire detection
  • live thermal camera POV
  • optimal intervention route calculation
  • displays road obstacles
  • displays fire hydrant locations
  • displays gas, electricity, water and sewage lines

Holistic was a EU backed project that I worked on during my employment at the Gemma laboratory It is a live map of the town Ajdovščina (Slovenia) and the sorrounding forests. The live feed is paired with a thermal camera and a drone surveiling the area in order to detect forest fires. The application alerts the local fire squad of any detected fires and calculates the optimal intervention route for them (considering current road obstacles). The map can also display fire hydrant, gas, electricity, water, and sewer line locations.

The application was developed with Java using Google Web Toolkit. The map interface is built with the OpenLayers library that recieves map data from a server running the open source GeoServer software.
