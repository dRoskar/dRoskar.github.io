BGC Ranker

Tool for ranking members of a board game club

Source Code

Features

  • make ranking lists for multiple games
  • add players
  • configure ranking ladders
  • set level-up formula
  • add rank images
  • player overview
  • keep track of player's victories
  • see when players level up

BGC Ranker is a tool I created back when I was running a Board Game club with my friends. We decided to add ranks to some of the games we played, to add some excitement and competition. Players would start on level one, and rank up to higher levels as they gained victories. Each rank would have a flair associated with it. We did this with a list at first but it became tedious, so I decided to create a tool that would help me calculate and keep track of people's rankings efficiently.
